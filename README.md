# LegalTrack

Fullstack legal case management system built with Spring Boot and React.

## 🚀 Overview

LegalTrack is a modern web application designed to manage legal cases, clients, and documents efficiently. It aims to simulate a real-world SaaS platform, providing a clean architecture and scalable structure.

This project is built to showcase fullstack development skills using modern technologies.

---

## 🛠️ Tech Stack

### Frontend
- React
- TypeScript
- Vite
- PNPM

### Backend (planned)
- Spring Boot
- Java
- REST API
- JWT Authentication

### Database (planned)
- PostgreSQL / MySQL

---

## ✨ Features (in progress)

- 🔐 User authentication (JWT)
- 👤 Client management
- 📁 Case management
- 📄 Document upload & management
- 📊 Dashboard with statistics
- 📅 Appointment system
